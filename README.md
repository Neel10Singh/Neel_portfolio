## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   https://github.com/Neel10Singh/Neel_portfolio
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
